# Time Series Analysis & Forecasting Web App

This project is a Python-based interactive web application for **time series analysis and forecasting** using real-world financial and cryptocurrency data.  
It provides multiple forecasting models and visual insights through a simple and intuitive **Streamlit** interface.

---

## Key Features

- Fetches real-time financial data using Yahoo Finance
- Cleans and preprocesses time series data
- Supports multiple forecasting techniques:
  - ARIMA (statistical model)
  - Prophet (trend & seasonality-based model)
  - Machine Learning–based forecasting
- Interactive data visualization
- User-friendly web interface using Streamlit

---

## Tech Stack

- **Language:** Python 3.12
- **Framework:** Streamlit
- **Libraries:**  
  - NumPy, Pandas – data handling  
  - Matplotlib, Seaborn, Plotly – visualization  
  - Scikit-learn – ML utilities  
  - Statsmodels – ARIMA  
  - Prophet – time series forecasting  
  - yfinance – real-time data collection  

---

## How to Use

Select a financial instrument (stock or crypto)
Choose a forecasting model
View historical trends and future predictions
Analyze patterns and insights through visual charts

---

## Project Objective

The goal of this project is to demonstrate time series analysis concepts and apply them to real-world financial data using both statistical and machine learning approaches, presented through an interactive dashboard.
